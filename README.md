byr-location
============

A Chrome extension that shows location of every post in BYR bbs. Available on [Chrome Web Store](https://chrome.google.com/webstore/detail/byr-ip-to-location/fkgdjdphlpgcccgfaacjkchmphafhcmd) ã€‚

IP information comes from [pytool.sinaapp.com](http://pytool.sinaapp.com) (source code: [JohnWong/python-tool](https://github.com/JohnWong/python-tool) @ Github</a>)

- http://bbs.byr.cn

- http://bbs.byr.edu.cn
- http://bbs6.byr.edu.cn
- http://forum.byr.edu.cn
- http://www.newsmth.net
- http://m.newsmth.net
